1. Factor97.csv
A comprehensive list of 97candidate factors.
2. Factor_des.csv
A reference file containing factor acronyms and their corresponding long-form descriptions.
3. Fama285.csv
A set of 285 bivariate-sorted basis portfolios from Ken French's website. The data in Fama285.csv has already subtracted the risk-free rate.
4. RP-PCA
Risk-premia-PCA (RP-PCA) factors proposed by Lettau and Pelger (2020b).
5. RRA
Reduced-rank approach (RRA) factors proposed by He, Huang, Li, and Zhou (2023).
